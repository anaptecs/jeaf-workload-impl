# JEAF Workload Impl README #

This repository contains the source code of JEAF Workload Impl. Project is an implementation of JEAF's Workload API. This implementation is intended to be used in high load production environments.

## Links ##
For further information please refer to:

* [JEAF Workload Management](https://anaptecs.atlassian.net/l/c/FPsTafiT)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.workload/jeaf-workload-impl)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.workload/jeaf-workload-impl/latest/index.html)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
